Question-2: What is the difference between innerHTML, innerText, and textContent ?
innerHTML হলো : যা HTML সহ সেট করে বা রিটার্ন করে। innerText হলো : যা কোন এলিমেন্টের মধ্যে থাকা text কে সেট বা রিটার্ন করে। textContent হলো : কোন এলিমেন্ট ডিসপ্লে নান থাকা সত্তেও তার মধ্যে থাকা text কে রিটার্ন করে এবং সেট করে
Question-3: What is event delegation in the DOM?
Event delegation হলো: Event delegation এমন একটি কৌশল যার মাধ্যমে পেরেন্ট এর মধ্যে Event Listener দিয়ে তার Child গুলোর মধ্যে Event পরিচালনা করা যায়
Question-4: What is event bubbling in the DOM?
Event bubbling : Event bubbling মুলত বিতর থেকে বাহিরের দিকে যায় বা উপরের দিকে উঠে অর্থাৎ কোন ইউজার যদি কোন বাটন এ ক্লিক করে তখন ব্রাউজার বুজতে পারে কোথাও ক্লিক কারা হয়েছে, যে স্থানে ক্লিক করা হয়েছে ঔ স্থানে ট্রিগার হয় তারপর সে স্থান বা সে বাটন এর পেরেন্ট এর কাছে যায় সেখানে কোন addEventListener থাকলে তাকেও ট্রিগার করে তার পর তারও পেরেন্ট এর কাছে যায় ওইখানে Event থাকলে ওইটাকেও ট্রিগার করে এইভাবে Document পর্যন্ত যায়।
